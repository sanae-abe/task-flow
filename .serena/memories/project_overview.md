# TaskFlow プロジェクト概要

## プロジェクトの目的

React + TypeScript で構築されたモダンなタスク管理アプリケーション「TaskFlow」

## 主要機能

- **タスク管理**: 作成・編集・削除・完了機能、期限・時刻設定、繰り返し設定
- **サブタスク**: チェックリスト形式の進捗管理
- **ラベル機能**: カスタムカラーラベルによる分類
- **ファイル添付**: ドラッグ&ドロップ対応（5MBまで）
- **複数ビュー**: カンバン・テーブル・カレンダービュー
- **フィルタリング・ソート**: 多角的なタスク整理
- **データ管理**: ローカル保存・インポート機能

## アーキテクチャパターン

- Context API による状態管理
- Component-based 設計
- Custom Hooks によるロジック再利用
- TypeScript Strict モード
- Primer React デザインシステム採用

## デプロイメント

- ブラウザのlocalStorageでオフライン対応
- HashRouter採用でS3静的ホスティング対応
- レスポンシブデザイン（モバイル・デスクトップ対応）
