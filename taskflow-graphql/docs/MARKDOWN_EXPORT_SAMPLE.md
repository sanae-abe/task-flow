## Usage Examples

### Query: Export Board as Markdown

```graphql
query {
  exportBoardAsMarkdown(boardId: "board-1")
}
```

### Query: Export Single Task

```graphql
query {
  exportTaskAsMarkdown(taskId: "task-123")
}
```

### Query: Export Filtered Tasks

```graphql
query {
  exportTasksAsMarkdown(
    boardId: "board-1"
    filters: {
      priority: HIGH
      status: TODO
    }
  )
}
```

### Mutation: Generate Markdown Report

```graphql
mutation {
  generateMarkdownReport(
    input: {
      boardId: "board-1"
      format: GITHUB_FLAVORED
      includeCompleted: true
      includeSubtasks: true
      includeLabels: true
      includeAttachments: true
      filters: {
        priority: HIGH
      }
    }
  ) {
    content
    generatedAt
    format
    metadata {
      boardName
      taskCount
      completedCount
      includeSubtasks
      includeLabels
      includeAttachments
    }
  }
}
```

## Features Demonstrated

1. **Task Checkboxes**: `[ ]` for incomplete, `[x]` for completed
2. **Priority Badges**: Visual indicators with emojis (ğŸ”´ğŸŸ ğŸŸ¡ğŸŸ¢)
3. **Labels**: Displayed as tags (#Security, #Backend, etc.)
4. **Due Dates**: Formatted dates with overdue warnings
5. **Subtasks**: Nested checkbox lists
6. **Attachments**: File listings with sizes
7. **Statistics**: Task counts, completion rates, priority breakdown
8. **Multiple Formats**: Standard, GitHub Flavored, and Obsidian
9. **Filtering**: By status, priority, labels, search query
10. **Metadata**: Board info, generation timestamp, task counts
