# 優先度システム ビジュアルガイド

## 改善前後の比較

### 1. PriorityBadge コンポーネント

#### 改善前
```
┌─────┐  ┌─────┐  ┌─────┐
│ 高  │  │ 中  │  │ 低  │
└─────┘  └─────┘  └─────┘
  赤      オレンジ   グレー

- テキストのみ
- 色だけで識別
- ホバー効果なし
- サイズ固定（small/large のみ）
```

#### 改善後
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ 🔥 高    │  │ ⚫ 中    │  │ ー 低    │
└──────────┘  └──────────┘  └──────────┘
  オレンジ      ブルー       グレー

- アイコン + テキスト
- 色 + 形で識別可能（アクセシビリティ）
- ホバー時スケール拡大（1.05倍）
- 3つのサイズ（small/medium/large）
- 3つのバリアント（filled/outlined/subtle）
```

### 2. カラーパレット

#### 改善前
| 優先度 | 背景色 | テキスト色 |
|--------|--------|------------|
| 高     | #cf222e (赤) | #ffffff |
| 中     | #fb8500 (オレンジ) | #ffffff |
| 低     | #656d76 (グレー) | #ffffff |

**問題点:**
- 高優先度が赤色（エラーとの混同）
- 中優先度と高優先度の色が近い

#### 改善後
| 優先度 | 背景色 | テキスト色 | ボーダー |
|--------|--------|------------|----------|
| 高     | #fb8500 (オレンジ) | #ffffff | #9a6700 |
| 中     | #0969da (ブルー) | #ffffff | #0550ae |
| 低     | #656d76 (グレー) | #ffffff | #57606a |

**改善点:**
- 高優先度はオレンジ（重要性を表現）
- 中優先度はブルー（GitHubブランドカラー）
- 明確な色の区別
- WCAG 2.1 AA準拠（コントラスト比 4.5:1以上）

### 3. アイコンシステム

#### アイコンの選定理由

**高優先度 - 🔥 Flame（炎）**
```
意味: 高い重要性、熱量、緊急性
心理: アクティブ、エネルギッシュ
用途: 近日中に対応が必要なタスク
```

**中優先度 - ⚫ DotFill（ドット）**
```
意味: ニュートラル、標準、バランス
心理: 落ち着き、安定
用途: 通常の優先度のタスク
```

**低優先度 - ー Dash（ダッシュ）**
```
意味: 控えめ、低い重要性
心理: 穏やか、待機
用途: 時間があるときに対応するタスク
```

### 4. バリアント比較

#### Filled（塗りつぶし）
```
使用場面: デフォルト、最も目立たせたい場合
視覚的特徴: 背景色が塗りつぶされている

┌──────────────┐
│ 🔥 高        │  ← 背景: #fb8500
└──────────────┘    テキスト: #ffffff
```

#### Outlined（アウトライン）
```
使用場面: 控えめに表示したい場合
視覚的特徴: ボーダーのみ、背景透明

┌──────────────┐
│ 🔥 高        │  ← 背景: transparent
└──────────────┘    ボーダー: #9a6700
                    テキスト: #fb8500
```

#### Subtle（微妙）
```
使用場面: 背景に溶け込ませたい場合
視覚的特徴: 薄い背景色

┌──────────────┐
│ 🔥 高        │  ← 背景: #fff8c5（薄いオレンジ）
└──────────────┘    テキスト: #9a6700
```

### 6. PrioritySelector コンポーネント

#### 改善前
```
┌─────────────────────┐
│ 高                ▼ │
└─────────────────────┘

クリック時:
┌─────────────────────┐
│ 選択なし            │
│ 高                  │
│ 中                  │
│ 低                  │
└─────────────────────┘

- シンプルな選択肢
- アイコンなし
- 説明なし
```

#### 改善後
```
┌─────────────────────┐
│ 🔥 高             ▼ │  ← 選択状態のプレビュー
└─────────────────────┘

クリック時:
┌───────────────────────────────┐
│ ⭕ 選択なし                    │
│   優先度を設定しない           │
├───────────────────────────────┤
│ 🔥 高                        1 │
│   近日中に対応が必要           │
├───────────────────────────────┤
│ ⚫ 中                        2 │
│   通常の優先度                 │
├───────────────────────────────┤
│ ー 低                        3 │
│   時間があるときに対応         │
└───────────────────────────────┘

- アイコン + ラベル + 説明
- キーボードショートカット表示
- 選択状態のハイライト
- ホバー/フォーカス効果
```

### 7. インタラクション詳細

#### ホバー効果
```css
transform: scale(1.05);
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);
transition: all 0.2s ease-in-out;
```

**視覚的効果:**
```
通常時:
┌──────────┐
│ 🔥 高    │
└──────────┘

ホバー時:
  ┌──────────┐
  │ 🔥 高    │  ← わずかに拡大
  └──────────┘
     影が表示
```

#### フォーカス状態
```css
border-color: #0969da;
box-shadow: 0 0 0 3px rgba(9, 105, 218, 0.3);
```

**視覚的効果:**
```
┌──────────┐
│ 🔥 高    │  ← 青い輪郭が表示
└──────────┘
   3px のフォーカスリング
```
